# TryHackMe Cybersecurity Labs

This repository contains all practical labs and exercises completed on TryHackMe to strengthen foundational cybersecurity skills. It is designed to be **flexible**, so new labs and exercises can be added over time without modifying the main structure.

---

## 🔹 Repository Overview

- **Purpose:** Showcase hands-on experience in Cybersecurity, Networking, Linux, Python, and Security fundamentals.
- **Content:** Labs, exercises, notes, and screenshots from TryHackMe.
- **Skills Developed:**  
  - Cybersecurity basics & defensive practices  
  - Phishing detection & social engineering awareness  
  - Networking concepts & traffic analysis  
  - Linux commands & system fundamentals  
  - Python scripting for small automation tasks
